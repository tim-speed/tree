Examples

Gangnam...sorry, Google style

Tree can optionally sort and order recursively or non-recursively by item type and title ascending or descending.

<ul class="nav nav-tree" role="tree">
  <li role="treeitem">
    <label>
      <b class="tree-item-toggle" role="presentation"></b>
      <i class="tree-item-icon icon-file"></i>
      <span class="tree-item-label">Item 1</span>
      <b class="selection-highlighter"></b>
    </label>
  </li>
  ...
  <li role="treeitem" aria-expanded="true" class="selected">
    <label>
      <b class="tree-item-toggle" role="presentation"></b>
      <i class="tree-item-icon icon-folder-open"></i>
      <span class="tree-item-label">Item 3</span>
      <b class="selection-highlighter"></b>
    </label>
    <!-- Nested Tree (Level 2) -->
    <ul role="group">
      <li role="treeitem">
        <label>
          <b class="tree-item-toggle" role="presentation"></b>
          <i class="tree-item-icon icon-file"></i>
          <span class="tree-item-label">Item 3.1</span>
          <b class="selection-highlighter"></b>
        </label>
      </li>
    </ul>
  </li>
  ...
</ul>
